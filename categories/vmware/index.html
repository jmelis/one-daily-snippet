generate and deploy OVAs

vmware

The sky is blue, the grass is green, and nobody likes the vCenter integration plugin. So let’s manage OVFs from the command line using ovftool.

Creating an OVA:

ovftool "vi://<user>:<password>@<host>/<path>/Datacenter/vm/vOneCloud-2.0.0-v5" myova.ova

Deploying an OVA:

ovftool -ds=<datastore> -n="<VM Name>" myova.ova "vi://<user>:<password>@<host>/<path>/Datacenter/host/Cluster"
