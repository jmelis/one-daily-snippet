a better onevm list

By Jaime Melis cli

This is a more useful onevm list:

alias onevmll="onevm list -l ID,NAME,IP,STAT --csv|sed 1d|column -ts,"

Example:

ip to hex

By Jaime Melis cli

In order to convert MAC to IP and back, use the ip_to_hex tool:

#!/usr/bin/env ruby

MAC_PREFIX = "02:00"

arg = ARGV[0]

if arg.include? ':'
    puts arg.split(':')[2..-1].collect{|i| i.to_i(16)}.join('.')
else
    mac = MAC_PREFIX + ':' + arg.split('.').collect{|i| sprintf("%02X",i)}.join(':')
    puts mac
end

Usage:

➜  ~  ip_to_hex 192.168.0.10
02:00:C0:A8:00:0A
➜  ~  ip_to_hex 02:00:C0:A8:00:0A
192.168.0.10

generate /etc/hosts

By Jaime Melis cli

If you ever need to generate quickly a snippet for /etc/hosts

onevm list -l ID,NAME,IP,STAT --csv | sed 1d | column -ts, | tr -d '"' | awk '{print $3, $2}' | column -t

Example:

10.10.0.37   migrator-4.14.2.devel
10.10.0.108  ubuntu1404-upgrade-d1395.test
10.10.0.116  cangallo
10.10.0.254  build-centos7-jaime.test
10.10.0.57   ubuntu1604-upgrade-cf116.test
