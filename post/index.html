generate and deploy OVAs

vmware

The sky is blue, the grass is green, and nobody likes the vCenter integration plugin. So let’s manage OVFs from the command line using ovftool.

Creating an OVA:

ovftool "vi://<user>:<password>@<host>/<path>/Datacenter/vm/vOneCloud-2.0.0-v5" myova.ova

Deploying an OVA:

ovftool -ds=<datastore> -n="<VM Name>" myova.ova "vi://<user>:<password>@<host>/<path>/Datacenter/host/Cluster"

onepry

By Jaime Melis cli

We can fire up pry with the OpenNebula libraries to debug it more easily:

#!/usr/bin/ruby

ONE_LOCATION=ENV["ONE_LOCATION"]

if !ONE_LOCATION
    RUBY_LIB_LOCATION="/usr/lib/one/ruby"
    VAR_LOCATION = "/var/lib/one"
    LIB_LOCATION = "/usr/lib/one"
    ETC_LOCATION = "/etc/one"
else
    RUBY_LIB_LOCATION=ONE_LOCATION+"/lib/ruby"
    VAR_LOCATION = ONE_LOCATION+"/var"
    LIB_LOCATION = ONE_LOCATION+"/lib"
    ETC_LOCATION = ONE_LOCATION + "/etc"
end

$: << RUBY_LIB_LOCATION
$: << RUBY_LIB_LOCATION+"/cloud" # For the Repository Manager
$: << RUBY_LIB_LOCATION+"/cli" # For the Repository Manager
$: << LIB_LOCATION+'/oneflow/lib'

################################################
# Required libraries
################################################
require 'yaml'
require 'erb'
require 'tempfile'
require 'fileutils'
require 'json'

require 'opennebula'
include OpenNebula

require 'pp'

@client = Client.new

module OpenNebula
    class XMLElement
        def self.from_xml(s)
            m = s.match(/^<(\w+)>/)
            root_element = m[1]

            elem = XMLElement.new
            elem.initialize_xml(s, root_element)
            return elem
        end
    end
end

version = ">= 0"
gem 'pry', version
load Gem.bin_path('pry', 'pry', version)

ip to hex

By Jaime Melis cli

In order to convert MAC to IP and back, use the ip_to_hex tool:

#!/usr/bin/env ruby

MAC_PREFIX = "02:00"

arg = ARGV[0]

if arg.include? ':'
    puts arg.split(':')[2..-1].collect{|i| i.to_i(16)}.join('.')
else
    mac = MAC_PREFIX + ':' + arg.split('.').collect{|i| sprintf("%02X",i)}.join(':')
    puts mac
end

Usage:

➜  ~  ip_to_hex 192.168.0.10
02:00:C0:A8:00:0A
➜  ~  ip_to_hex 02:00:C0:A8:00:0A
192.168.0.10

git prepare commit msg

By Jaime Melis git

Forget about adding feature #blablabla to every single commit! and it’s free!

#!/bin/sh

BRANCH_NAME=$(git symbolic-ref --short HEAD)

if [[ $BRANCH_NAME != bug-* ]] && [[ $BRANCH_NAME != feature-* ]]; then
    exit 0
fi

FIRST_LINE_COMMIT=$(head -n1 $1)

case $FIRST_LINE_COMMIT in
bug*|Bug*|feature*|Feature*)
    exit 0
    ;;
esac

LABEL=$(echo $BRANCH_NAME|sed 's/-/ #/')
LABEL=${LABEL^}

sed -i -e "1s/^/${LABEL}: /" $1
